# AI2_cercles
Trois cercles imbriqués pour tracer des courbes

Le premier cercle est centré sur le canvas. Le second a son centre sur le rayon du premier et le troisième sur le rayon du deuxième.

Les rayons se font par poisitinnement des trois sprites qui marquent chaque cercle. Les rayons ont des rapports entiers pour que les courbes se chevauchent à chaque tour

Des sliders permettent de changer la vitesses de rotation de chaque cercle.
